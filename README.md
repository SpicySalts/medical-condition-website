# medical-condition-website
HTML/CSS website documenting asthma

## Project Overview
This website displays information about asthma. It features responsive design optmized for mobile devices, with text, images, and tables adjusting to different resolutions in real time.

## Responsive Design:
- Text, images, and tables adjust proportionally to the resolution
- Tables and lists fit within the screen without horizontal scrolling
- Images stack vertically on small screens and remain aligned on larger screens

## Key Features:
- Three images each one being centered, left=aligned, or right-aligned
- Ordered lists and a table with rowspan/colspan
- Responsive design for mobile devices
- CSS Styling

## How to Run

1. Download or clone the repository.  
2. Open the project folder.  
3. Open the `index.html` file in any web browser
